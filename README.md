# Wechat-Cleanmaster
A new generation clean tool for new wechat file system

本脚本专清微信Msgattach文件夹，现在所有缓存文件（图片、视频、音频以及接收的文件）都存在这里，本脚本将会将接收的文件移动到以日期命名的文件夹中，其余的进行删除处理。

本脚本还在测试中，请谨慎使用。

欢迎commit！
